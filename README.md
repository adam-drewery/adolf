## Adolf.NET: Azure DevOps Lightweight Feed

#####A terminal client for Azure DevOps

---

#### Authentication
`adolph auth 
    -url "https://your-url-here.visualstudio.com/"
    -token "your-pat-token-here"`
    
#### View Work Item
`adolph work 12345` to view work item #12345
